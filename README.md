# 💰 Finance Dashboard (Frontend)

A modern role-based Finance Dashboard built with **React.js**, featuring authentication, protected routes, and role-based access control (Admin, Analyst, Viewer).

---

## 🚀 Features

- 🔐 Authentication (Login with JWT)
- 🧑‍💼 Role-Based Access Control (RBAC)
  - Admin → Full access
  - Analyst → Dashboard + Records
  - Viewer → Read-only (Records only)
- 📊 Dashboard (Income, Expense, Balance)
- 📑 Records Management (CRUD)
- 👥 Users Management (Admin only)
- 📱 Fully Responsive UI (Custom CSS)
- 🔒 Protected Routes
- ⚡ Axios API Integration with Interceptors
- 💾 Persistent Login (localStorage)

---

## 🛠 Tech Stack

- React.js (Vite)
- React Router DOM
- Axios
- Context API (Auth Management)
- Custom CSS (No Tailwind)
